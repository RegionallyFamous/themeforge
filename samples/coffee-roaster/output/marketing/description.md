# Bellwether — Coffee Roaster Theme

A heritage-leaning, editorial WooCommerce block theme built for
small-batch specialty coffee roasters. Six style variations, polished
placeholder system, and demo content ready to import.

## What's inside

- Full Site Editing (FSE) block theme with editor-managed layouts.
- WooCommerce-ready: product catalog, single product, cart, checkout,
  account — all using the official WooCommerce blocks.
- Six style variations: Heritage (default), Editorial, Mono, Sage,
  Dark, and Field. Switch between them in two clicks.
- Newsletter, FAQ, testimonial, and "About the roast" patterns.
- One-click demo content import (products, blog posts, sample reviews).
- Fluid typography and spacing scales — looks right on every screen.
- Theme.json-driven, no PHP customizations needed.

## Built for

Small-batch coffee roasters, single-origin importers, espresso bars
selling beans online, and subscription roasters. Optimized for
storefronts where the buyer reads before they buy.

## Compatibility

- WordPress 6.5+
- WooCommerce 8.5+
- PHP 7.4+

## Variations

- **Heritage** — warm browns, serif headlines, generous spacing.
- **Editorial** — high-contrast, larger type scale, magazine layouts.
- **Mono** — single accent, restraint, gallery-style product grids.
- **Sage** — muted greens, botanical, soft.
- **Dark** — same shape, low-light palette.
- **Field** — earthier, wider gutters, agricultural reference.
